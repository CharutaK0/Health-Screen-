# Health-Screen-
Health Screen is an AI assisted, preliminary deficiency detection tool that can be used by medical professionals, and the everyday curious mind. It is a "SYMPTOM- BASED" screening tool, hence it dose not use any blood-test values, or Vitamin intake values to come to a suggestion.It does not conclude a confirmed diagnosis, but only suggests the probability of a deficiency being present based on a Questionnaire that is answered by the user. It is an assistive tool. Blood-test values, and Vitamin intake values etc. are part of the future expansion scope of this project.
